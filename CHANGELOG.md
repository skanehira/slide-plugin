# CHANGELOG

## [1.2.0] - 2026-07-01

### Added

**プラグイン配布対応**
- `.claude-plugin/plugin.json` を追加し、Claude Code の正式なプラグイン形式に対応
- `.claude-plugin/marketplace.json` を追加し、このリポジトリ自身をマーケットプレイスとして自己完結させ、`/plugin marketplace add skanehira/slide-skill` → `/plugin install slide-plugin` でインストール可能に

### Changed

**`slide-deck-builder` の自動フォールバック**
- カレントディレクトリに `SLIDE-md/` が無い場合、プラグイン同梱の `docs/SLIDE-md/`（10種類）に自動フォールバックし、選択したデザインシステムをカスタマイズ起点としてプロジェクトへコピーするように変更
- カレントディレクトリに `SLIDE-PATTERN/` が無い場合、プラグイン同梱の `docs/SLIDE-PATTERN/`（99種類）を直接参照するように変更（パターンは読み取り専用の共有資産のためコピーはしない）
- 同梱ファイルも見つからない場合は、`slide-md-creator` / `slide-pattern-creator` の実行要否をその場で確認し、承認されればスキルを自動的に呼び出して続行するように変更（従来はユーザーへの案内メッセージを出してスキルを終了していた）

**README**
- インストール手順を `/plugin marketplace add` → `/plugin install` によるプラグインインストール方式に一本化（従来の手動コピー手順（方法A・方法B）を削除）

## [1.1.1] - 2026-06-29

### Added

**サンプルデザインシステムの拡充（docs/SLIDE-md/）**
- サンプルを 4 種類 → 10 種類に拡充
- 新規追加した 6 種類：
  - `SLIDE-md-golden-yellow`（ゴールデンイエロー：BtoBビジネス発表・社内共有）
  - `SLIDE-md-blue-teal-recruitment`（ブルー×ティール：採用・コーポレートブランディング）
  - `SLIDE-md-blue-simple-diagram`（ブルー：教育・研修・図解型プレゼン）
  - `SLIDE-md-green-blue-business`（グリーン×ブルー：営業・サービス提案・BtoB）
  - `SLIDE-md-corporate-red`（コーポレートレッド：決算説明会・IR資料）
  - `SLIDE-md-sky-corporate`（スカイコーポレート：統合報告書・アニュアルレポート）
- 全サンプルのメタ情報（表示名・想定シーン・カラースウォッチ）を管理する `index.json` を追加

### Changed

**既存 4 種類のサンプルを全面改訂**
- `SLIDE-md-MintGreen`・`SLIDE-md-anthropic`・`SLIDE-md-digital`・`SLIDE-md-gemini-color-system` を v1.1.0 の sample.html 共通実装スタイルで作り直し
  - 背景アクセント丸・縦バー・ページ番号・ブランドフッター・カードコンポーネントの標準スタイルを適用
  - スライドサイズ：960 × 540 px → 1920 × 1080 px（0.75 倍スケール表示）に統一

## [1.1.0] - 2026-06-28

### Added

**デザイントークンの拡張**
- カラーに `Text Sub`（見出し・ラベル色）と `Text Muted`（補足・キャプション色）を追加
- カラーに `Gradient`（グラデーションの有無・カラーストップ・方向）を追加
- タイポグラフィに `モノスペースフォント（Mono）` を追加（数値・コード・ページ番号に使われるフォント）

**新セクション**
- `Background Accent（背景装飾）`：半透明の装飾円・装飾図形の有無・配置・カラーを定義するセクションを追加
- `Component Style（コンポーネントスタイル）`：カード角丸・影・縦バー・箇条書きマーカー・番号スタイルを定義するセクションを追加

**Slide Frame の拡張**
- ページ番号スタイル（グラデーションテキスト / 単色・モノスペースフォント使用有無）を追加
- ブランドフッター（左下表示・内容）を追加
- 背景アクセント（半透明グラデーション丸の有無・配置・カラー）を追加
- 縦バー（見出し左側の縦バーの有無・スタイル・幅）を追加

**sample.html 共通実装**
- 全ページ共通の装飾要素として、背景アクセント丸・縦バー・ページ番号・ブランドフッター・カードコンポーネントの標準スタイルを追加
- ダミーコンテンツポリシーを策定（会社名は「サンプル株式会社」に統一、本文・数値はすべて汎用ダミーデータを使用）

### Changed

**デザイントークン**
- 見出しフォントの役割ラベルを「見出しフォント」→「見出しフォント（Display）」に変更
- 本文フォントの役割ラベルを「本文フォント」→「本文フォント（Body）」に変更

**sample.html 生成仕様**
- スライド数：5ページ → 6ページ
- スライドサイズ：960 × 540 px → 1920 × 1080 px（ブラウザ上では0.75倍スケール＝1440 × 810 px相当で表示）
- ページ1（仕様書ページ）を全面改訂：カラーパレット・スライドで使うパーツ・タイポグラフィ・レイアウト・Do/Don'tを1画面に集約
- ページ4：箇条書きをグラデーション円の番号付きカード形式に変更
- ページ5：棒グラフ（左）＋ドーナツグラフ（右）の2グラフ構成に変更
- ページ6：3列カード（価値と説明）＋締めのCTAボタン構成に変更
- CSSカスタムプロパティに `--color-text-sub` / `--color-text-muted` / `--gradient-main` / `--gradient-vert` / `--font-mono` 等を追加
